## Installation
```bash
    pip install mdml_client
```

## Documentation

Check out our [Read the Docs](https://mdml-client.readthedocs.io/en/latest/index.html#)

There is also a Jupyter Notebook inside the tutorial folder that demonstrates basic usage of the MDML client. There are two main components to the client: a producer class and a consumer class.
